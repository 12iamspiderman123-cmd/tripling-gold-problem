The Problem

There is a magic bag that increases the number of coins you put inside. If you put coins in, take them out, and then put that new amount back in a second time, you will end up with exactly triple your original starting amount.
1. Coins always increase.
2. The bag function is constant. If you put x coins in and y coins come out, this will always remain the same. Whenever x coins are placed in the bag, y coins will come out.
3. The more gold goes in, the more gold comes out.
4. Coins always go in or come out in natural numbers. No fractions are involved.

Question 1 (Beginner)

Determine the number of coins that will come out immediately if 13 coins are put in the bag.

Question 2 (Advanced)

Generate a function f(x) that provides the coins that come out immediately after x coins go in.

Solution to Question 1

“For any confusion, refer to the YouTube video. Use the table provided on this page to ensure that your data is correct. Writing entries is a very helpful approach for such kinds of problems. So we begin by writing entries from the beginning. We will be able to create a table-like structure that contains three columns. The first column will be Input, the second column will be Intermediate, and the third column will be Output. The third column is obviously 3 times the input. Now we will start by writing our very first input, which is one coin. If one coin goes into the bag, let’s say y coins come out. If these y coins go into the bag again, 3 coins come out. We all know that this bag sets some kind of constraints. One of them is that a greater number of coins must always come out. Using this, we can clearly say that y must be greater than 1. Applying the same logic again, we can say that 3 must be greater than y. Now if we solve these two inequalities and apply the constraint that y must be a natural number, we get that y must be 2 and it cannot be anything else. Moving forward, we can now say that if 2 coins go into the bag, 3 coins must come out, as it was the result of our very first entry. So now we have a set of entries. First entry: If 1 coin goes in, 2 coins come out. If those 2 coins go in again, 3 coins come out. Second entry: If 2 coins go in, 3 coins come out. If those 3 coins go in, 6 coins come out. Now for the third entry, we can clearly say that if 3 coins go in, 6 coins come out, and if those 6 coins go in, 9 coins come out. If we try to repeat this process, we will hit a dead end at Entry 4. That is because 4 has never appeared in any previous entries. We know what will come out if 2, 3, or 6 coins go in, but we do not know about 4. So now we are going to use a very clever trick. We will leave 4 and 5 aside and jump directly to the 6th entry. Then we write: If 6 goes in, 9 comes out. If 9 goes in, 18 comes out. This might sound unusual and surprising, but now with this 6th entry we can definitely say what will come out if 4 or 5 goes in. Seems surprising, doesn’t it? Now we will use the third point, which stated that if more gold goes in, more gold comes out. Look at the table provided on this page and notice: When we placed 3 coins into the bag, 6 coins came out. When we placed 6 coins into the bag, 9 coins came out. Let us assume: If 4 coins go in, A coins come out. If 5 coins go in, B coins come out. Now we obtain a set of interesting inequalities. The first inequality states that A must be greater than 6. The second inequality states that B must be greater than A. The final inequality states that 9 must be greater than B. Using the information that A and B must be natural numbers, we can clearly say that A is 7 and B is 8. If you repeat this process again and again until you reach Entry 15, you will clearly have the answer to what amount of coins will come out if 13 coins go in. That answer is 22. I would strongly suggest watching the YouTube video to understand this even better. The table provided on this page is extremely helpful.”
